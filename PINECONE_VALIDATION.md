## pinecone validation

chunks-sync was validated against a real pinecone index using pinecone-hosted embeddings.

configuration:
- Vector DB: Pinecone
- Embedding model: llama-text-embed-v2
- Embedding dimension: 1024
- Namespace: chunks-sync-demo
